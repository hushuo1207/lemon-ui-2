#  柠檬UI  lemon-ui 一个Vue UI 组件



## 介绍
  这是我在学历 Vue 过程中做的一个 UI 框架， 希望对你有用

## 开始使用

1. 添加样式
    使用本框架前，请在CSS内开启 border-box
    ```
    *, *::before, *::after{box-sizing: border-box;}


    --font-size: 14px;
    --button-height: 32px;
    --button-bg: white;
    --button-active-bg: #eee;
    --border-radius: 4px;
    --color: #333;
    --border-color: #999;
    --border-color-hover: #666;
    ```
   IE 15 以上支持此样式
2. 安装lemongulu
   npm i --save lemongulu
